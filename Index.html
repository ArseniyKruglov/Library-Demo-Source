Совместимость

Библиотека полностью совместима с современными версиями Chromium, Firefox и Safari. И, судя по данным Can I Use, будет работать у более чем 95% пользователей. Основные ограничители — CSS-переменные и веб-компоненты.

Цвета

Основной цвет задается как CSS-переменные: Hue, Saturation и Lightness. На их основе создается цветовая палитра сайта.
:root
{
   --Hue: 0;
   --Saturation: 70%;
   --Lightness: 50%;
}
Серый — не серый — к нему подмешивается немного другого цвета.
:root
{
   --Gray-Hue: 210;
   --Gray-Saturation: 10%;
}

Шрифты

Все шрифты разделены на четыре категории: заголовочные, текстовые, интерфейсные и моноширинные.

Скругление

Задаётся как CSS-переменная.
