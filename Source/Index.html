Исходный код

Цвета

Основной цвет задается как CSS-переменные: Hue, Saturation и Lightness. На их основе создается цветовая палитра сайта.
:root
{
   --Hue: 0;
   --Saturation: 70%;
   --Lightness: 50%;
}
Серый — не серый — к нему подмешивается немного другого цвета.
:root
{
   --Gray-Hue: 210;
   --Gray-Saturation: 10%;
}

Шрифты

Все шрифты разделены на четыре категории: заголовочные, текстовые, интерфейсные и моноширинные.
--Font-Title: "Google Sans", #{$Font-Fallback-Sans-serif};
--Font-Text: "Lora", #{$Font-Fallback-Serif};
--Font-UI: "Google Sans", #{$Font-Fallback-Sans-serif};
--Font-Monospace: "Roboto Mono", #{$Font-Fallback-Monospace};
